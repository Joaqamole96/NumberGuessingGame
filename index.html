Number Guessing Game

Welcome to the random number guessing game!

  1. The system generates fifteen (15) random integers between 1 and 100.
  2. Enter your guess to check if it exists in the system's collection.
  3. If found, the game will show its index and the number itself.
  4. If not, the game will tell you it's not in the collection.
  5. Your total number of tries will be counted.
  6. You can reveal the random number collection or reset it anytime.

Try to find as many numbers as you can with the fewest tries!
